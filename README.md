# Jmc2025
「じんもんこん2025発表用公開資料集」

本リポジトリは、情報処理学会「人文科学とコンピュータ」シンポジウム2025における発表  
「古楽譜解読プロセスの可視化：琵琶譜『三五要録』による催馬楽旋律分析を例に」  
（発表者：本塚　亘）で用いたデータの一部を公開するものです。

## 作成者
本塚　亘

## 内容

- `tei/`  
  宮内庁書陵部所蔵資料『三五要録』にもとづく翻刻XML・整形XML（TEI準拠）。

- `xslt/`  
  TEI/XML から分析用データを抽出するための XSLT スタイルシート。

- `yaml/`  
  音価や unit に関する設定情報。分析処理から参照されることを想定したものです。

## Python スクリプトについて

XSLT変換・分析に用いた Python スクリプトは、本リポジトリの利用者の環境に与えうる影響や  
ファイル操作等に関するリスクへの配慮・注意喚起を，公開用として十分に整備できていないため、  
現時点では本リポジトリには含めていません。
